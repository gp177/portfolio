Library Database

Team Members: Giorgio Plescia, Winoto Janputra, Oleksii Redko, Mojgan Ahmadikoohbanani



A fully working Library Database, this database can enroll members as either juvenile or adult, as a juvenile reaches the age of 18 a trigger initiates which moves them to the adults table. There are procedures for reserving a book and determining the books availability, or searching for specific details (titles, authors, etc.). For checking out books, a view and a function are made to calculate and assign an expiry date to the members, while check out procedure initiates when a card is scanned for checkout. for checking in books, a trigger initiates for updating the loan history after it is brought back.


Technologies Used:

  • SQL Server
  • SQL Server Management Studio
