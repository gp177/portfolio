Travel Database

The Travel Database application manages a persons list of trips through a connect of the database. On the user interface, destination, name, passport number, departure and return date are added and saved to a database table. Once on the database the List reads from the table and displays the results in a String.


Technologies Used:

  • Java Swing
  • MYSQL

Notepad Application

The Notepad Application is a standard notepad which the user can type on and save the file with any extension, Or open another file to display the text on this application. There is also a font color option, which opens a color pallet that allows the user to select any premade colors, or create their own. There is a Word wrap option which the use can select or deselect with the checkbox. Proper validation has been implemented, The application detects if text has been modified and will prompt to save if trying to exit.


Technologies Used:

  • Java Swing

TodoList Application

The TodoList Application is a graphical user interface for people to submit information about their tasks, they are prompted to enter, task name, due date, and completion, which gets added to a database table. The user is able to Delete or Update the added record, which edits the database and displays the new results in the JList.


Technologies Used:

  • Java Swing
