Dating Website

Team Members: Giorgio Plescia, Oleksii Redko



A dating website filled with features where users can create an account with personal info (name, age, gender, sexual orientation, location, image, etc..), then search and connect with others of interest. They can initiate private chats with the other user, typing which initiates a live chat. The user pages will be functional where users can customize their profiles, adding photo albums, lengthy descriptions, details about themselves physically or mentally to make it more attractive to catch others attention. Users will have a ticket system where they can open a ticket, selecting a certain department and description for admins.
There is an admin panel where admins can see list of users in which they can Email warn or remove, they can view tickets opened by users, and admins can create a special token link for other admin invitation.
